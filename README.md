# 1. 项目介绍

express-generator 脚手架生成的项目。

# 2. 使用

## 2.1 安装依赖

```bash
npm i
```

## 2.3 运行项目

- ./bin/www 文件中的入口文件已切换为 `var app = require("../app.ts");`

- 运行下面的脚本指令：
  ```bash
  DEBUG=express-generator-demo:* npm run dev:ts
  ```
  或者：
  ```bash
  npm run dev:ts
  ```

# 3. 访问项目

访问 `http://localhost:3000`
